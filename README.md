# mosque-signage

A simple digital signage application to show prayer times in the mosque.

Currently supported devices: AndroidTV, RaspberryPi

Demo: https://metinkale38.github.io/mosque-signage/

## Android
_Suggested if you want to use a TV or a set-top box with Android TV OS_

Install the app from the Play Store: https://play.google.com/store/apps/details?id=org.metinkale.signage&



## Raspberry Pi
_Suggested if you want to use HDMI-CEC to control the power of the display_

1. Install pi-signage (https://github.com/metinkale38/pi-signage) with chromium mode
   2. Go To https://metinkale38.github.io/pi-signage/?ui=/mosque-signage to send your configuration to the RPI with WebBluetooth

Tested devices: Raspberry Pi Zero 2W (should work for most Raspberry PI models)
